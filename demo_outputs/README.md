# Demo Outputs

This folder contains sample output files generated by running the Insect Detection notebook. These serve as reference examples of what the project produces.

## Files

### Training History
- `vgg16_history.csv` — VGG16 model training metrics across 20 epochs
- `resnet50_history.csv` — ResNet50 model training metrics across 20 epochs

Columns:
- `epoch` — Training epoch number
- `accuracy` — Training accuracy for the epoch
- `loss` — Training loss for the epoch
- `val_accuracy` — Validation accuracy for the epoch
- `val_loss` — Validation loss for the epoch

### Performance Metrics
- `model_metrics.csv` — Performance comparison across all 4 models
  - Includes: Accuracy, Precision, Recall, F1-Score

- `ensemble_results.csv` — Performance of different ensemble methods
  - Soft Voting: Averages prediction probabilities
  - Hard Voting: Majority vote on predicted classes
  - Weighted Average: Best overall performance at 93.2% accuracy

### Visualizations
- `model_accuracy_comparison.png` — Bar chart comparing accuracy across the 4 models
- `confusion_matrix_demo.png` — Confusion matrix heatmap for demo predictions
- `training_curves.png` — Training accuracy and loss curves for the demo output

## How These Were Generated

1. Run `NEW_FINAL_VERSION_8_9_INSECT_DETECTION.ipynb`
2. Models train on your AGRO dataset
3. CSV files and visualizations are automatically saved
4. Copy outputs to `demo_outputs/` folder

## Reproducing Results

To generate your own results:
```bash
# 1. Prepare your dataset
AGRO/train/<class_name>/
AGRO/test/<class_name>/

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run notebook
jupyter notebook NEW_FINAL_VERSION_8_9_INSECT_DETECTION.ipynb

# 4. Copy generated files
cp *_history.csv demo_outputs/
cp *_metrics.csv demo_outputs/
cp *.png demo_outputs/
```
